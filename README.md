# Install requirements

```bash
$ sudo npm install -g gulp bower
$ npm install gulp bower del gulp-concat gulp-jshint gulp-uglify gulp-minify-css gulp-assets --save-dev
```

# Install libraries

```bash
$ composer install --no-dev --optimize-autoloader
```

# Install assets

```bash
$ npm install
$ bower install
```

# Create cache files

```bash
$ mkdir cache/
$ chmod 777 cache/
```
